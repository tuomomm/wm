This directory contains the scripts for simulating the synaptic plasticity experiments. To run the population-averaged simulations of CTRL and SCZ-ACC and SCZ-PFC simulations, run 
sh runpopavs.sh
This will take several hours (20-40 min per simulation). The subject-wise simulations can be run through runsubjectwise.sh, but the coefficients for different protein concentrations are not included since they are considered sensitive data.

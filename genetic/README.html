This directory contains the scripts for carrying out the genetic association analyses. Most data analysis files are dependent on the specific structure at the sensite data server (TSD). No data included here since they are sensitive. The scripts use various annotation files that are publicly available and summary statistics files.

To run the analyses, first run doextract.sh to retrieve the beginning and end base pairs of each gene of interest. Then run doextractSNPs.sh to get the relevant SNPs for each gene set and extract the data into separate files according to different SNP p-value thresholds. Then run doextractPRSs.sh to extract genetic data from your sample from the SNPs of interest and to form polygenic risk scores for the chosen gene sets. Finally, run doassociations.sh to associate the PRSs to your WM data.

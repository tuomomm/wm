This entry contains the network and single-cell simulation scripts for the article "Multimodal evidence for a mechanistic model of working memory deficits in schizophrenia". The simulations scripts are Python+NEURON scripts. Tested with Python 3.9.21 and NEURON v. 8.2.2. Some of the scripts also work with NEURON v.9.0.0 (the ACG and MCellRan4 mechanisms have to be replaced by Random123).

Most of the subject-wise data is not included since they are sensitive, but the CACNA1I data are included since the subjects are anonymized and only one data point is provided per subject.

To run the simulations, run the following scripts: 
 nrnivmodl            #Compile the required mechanisms
 sh runfig_methods.sh #Run single-cell and network simulations needed for the Methods illustration of the model. The single-cell simulations last < 1 hour in total (single-CPU), the network simulation takes a couple of hours when parallellized to 8 CPUs
 sh runfig1.sh        #Run single-cell simulations with CTRL and SCZ-ACC and SCZ-PFC variants
 sh runfig2.sh        #Run network simulations with CTRL and SCZ-ACC and SCZ-PFC variants, mostly genewise
 sh runfig3.sh        #Run network simulations with CTRL and SCZ-ACC and SCZ-PFC variants (including simulations where Ca transients are saved), mostly genewise
 sh runfig3b.sh       #Run network simulations with CTRL and SCZ-ACC and SCZ-PFC variants (with NMDA current recording)
 sh runfig6.sh        #Run personalized network simulations with CTRL and SCZ-ACC and SCZ-PFC variants of CACNA1I expression
 sh runfig_supp.sh    #Run supplementary

To draw the figures, run "python3 drawfigXXX.py", mostly without arguments (apart from drawfig4C-G_and_S7.py for which argument 'PFC' can be given to draw the alternative figure, Fig. S7). File breakdown:
